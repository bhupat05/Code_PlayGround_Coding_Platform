# Code-Playground-Coding-Plateform-
A coding platform web application

## How to Run Locally

1. **Install MongoDB Compass** and ensure it's running.
2. **Create a `.env` file** in the root directory:
   ```env
   PORT=4000
   MONGODB_URI=mongodb://127.0.0.1:27017/code_playground
   NODE_ENV=development
   ```
3. **Install dependencies**: run `npm install`
4. **Start the server**: run `npm start` (or `npm run dev` for auto-reload)
5. **View app**: open `http://localhost:4000` in your browser
